# Spotify Analytics Platform - Frontend

**Live Application:** [https://spotify-analytics-platform.vercel.app](https://spotify-analytics-platform.vercel.app)

This is the frontend for the Spotify Analytics Platform, built with React, Vite, and Tailwind CSS. It connects to our FastAPI backend to display data insights, machine learning predictions, and track recommendations.

## Running Locally

1. Create a `.env` file in this directory:
```
VITE_API_URL=http://localhost:8000
```

2. Install dependencies and start the dev server:
```bash
npm install
npm run dev
```
